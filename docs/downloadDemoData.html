download additional imgraft demo data

targetdir=downloadDemoData(package)

This code will download additional demo data from the imgraft.glaciology.net website provided that it has not already been downloaded.

inputs: Package can be: 'cias', 'practise', or 'imcorr'

outputs: targetdir is the location of the downloaded files.
